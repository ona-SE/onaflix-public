# Expense Tracking Dashboard

## What I Need

I need a simple web page that shows our team's expense data in a visual way. I'm not a developer — I just want to open a file in my browser and see charts.

## Data

The expense data is in `data/expenses.csv`. Each row has a date, category, vendor, amount, and description.

## What I Want to See

1. **Monthly totals** — a bar chart showing how much we spent each month
2. **Spending by category** — a pie or doughnut chart breaking down where the money goes (Travel, Software, Office Supplies, etc.)
3. **Top vendors** — a list or chart of the vendors we spend the most with
4. **Trend over time** — a line chart showing whether spending is going up or down month to month

## Requirements

- Must work by just opening `index.html` in a browser — no server, no install, no build step
- Should look clean and professional (not like a homework assignment)
- Use the CSV file in `data/expenses.csv` as the data source
- Include a summary section at the top with total spend, average monthly spend, and number of transactions
- Make it responsive so it looks OK on a laptop screen

## Nice to Have

- Ability to filter by category
- Color coding by category that's consistent across all charts
